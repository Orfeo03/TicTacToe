TicTacToe 

This is a simple Tic-Tac-Toe game made with Java and Swing.
Two players take turns playing on the same computer, with the game detecting wins and ties automatically.

Features

 -3x3 grid using buttons

 -Turn-based gameplay (X and O)

 -Win and tie detection

 -Clean and simple UI

How to Run

-Clone the repository

-Open it in IntelliJ IDEA or any Java IDE

-Run the TicTacToe class

How to Play

-Player X starts

-Click on an empty square to place your mark

-The game ends when a player wins or the board is full

-This project was built to practice Java Swing and event handling.
